C++ Projects:


Project : Comprehensive Currency Converter
Introduction:

I submitted it as my 2nd semester Programming Fundamental's project. This project implements a console-based Currency Converter program using C++. The program allows users to convert amounts from one currency to another using predefined conversion rates. The user can choose from a list of 20 currencies and perform multiple conversions, with the results being saved to a file for record-keeping.

Objective:

The primary objective of this project is to develop a user-friendly application that facilitates currency conversion between various international currencies. Additionally, the project aims to demonstrate the use of basic C++ features such as file handling, loops, arrays, and formatted output.

Key Features:
  • User Interface: A simple console-based UI for input and output.
  • Currency List: Supports conversion between 20 different currencies.
  • Data Persistence: Records conversion results in a text file.
  • Continuous Operation: Allows multiple conversions without restarting the program.
Implementation Details:
  • Welcome Screen: Displays a welcome message.
  • User Choice: User can choose to continue with the conversion or exit the program.
  • Currency Selection: User selects the source and target currencies from a list.
  • Amount Input: User inputs the amount to be converted.
  • Conversion Calculation: Program calculates the converted amount.
  • Output: The result is displayed on the console and saved to a file.
  • Repeat or Exit: User can choose to perform another conversion or exit.
Conclusion:

The Currency Converter project successfully demonstrates the use of fundamental C++ programming constructs to build a functional application. The program handles user input, performs calculations, manages file I/O, and provides a continuous user experience. This project can be further enhanced by integrating real-time currency exchange rates through an API and adding a more sophisticated user interface.

Project : Aurdino Based Ultrasonic Radar
Introduction:

I presented this project as part of the PGC Robotics Exhibition 2022. The project involves creating an ultrasonic radar system using an Arduino microcontroller. This radar system uses an ultrasonic sensor to detect objects within a specified range, displaying the detected object's distance and position on a graphical interface. The radar provides a real-time scanning capability, making it an effective tool for distance measurement and object detection.

Objective:

The primary objective of this project is to design and implement a radar system that can detect and locate objects within its range. The project aims to showcase the integration of hardware (Arduino and sensors) with software (data visualization) to create a functional and interactive system.

Key Features:
  • User Interface: A graphical interface for real-time display of detected objects.
  • Object Detection: Uses an ultrasonic sensor to detect objects within a specified range.
  • Distance Measurement: Calculates the distance to detected objects and displays it.
  • Real-Time Scanning: Continuously scans the environment and updates the display.
  • Data Visualization: Shows the position of detected objects on a graphical radar screen.
Implementation Details:
  • System Initialization: Initializes the Arduino, ultrasonic sensor, and display module.
  • Scanning Process: The ultrasonic sensor scans the environment within a specified range and angle.
  • Object Detection: Detects objects based on the reflection of ultrasonic waves.
  • Distance Calculation: Calculates the distance to the detected object using the time delay of the reflected wave.
  • Graphical Output: Displays the detected object's position on a graphical radar screen.
Conclusion:

The Arduino-Based Ultrasonic Radar project successfully demonstrates the integration of hardware components with software to create a functional radar system. The project handles object detection, distance calculation, and real-time data visualization. Future enhancements could include increasing the radar's detection range, integrating more sensors for better coverage, and improving the graphical interface for a more detailed display.

Project : Aurdino Based Bluetooth Supported Car
Introduction:

The Arduino-Based Bluetooth-Supported Car is a remote-controlled vehicle that can be operated using a smartphone. The car is equipped with an Arduino microcontroller and a Bluetooth module, allowing it to receive commands wirelessly. This project demonstrates the use of Arduino to create a functional and interactive robotic car that can be easily controlled via Bluetooth.This was a part of our projects in PGC Robotics Exhibition 2022.

Objective:

The primary objective of this project is to develop a Bluetooth-controlled car that can be maneuvered remotely using a smartphone. The project aims to showcase the integration of Bluetooth technology with Arduino to create a wireless control system. It also serves as an introduction to basic robotics and wireless communication.

Key Features:
  • User Interface: A mobile application interface for controlling the car's movements.
  • Wireless Control: Uses Bluetooth technology to receive and execute commands from a smartphone.
  • Directional Movement: Supports forward, backward, left, and right movements.
  • Real-Time Response: Executes commands with minimal delay for smooth operation.
  • Compact Design: The car is designed to be small and agile, making it easy to maneuver in tight spaces.
Implementation Details:
  • System Initialization: The Arduino initializes the Bluetooth module and motors.
  • Bluetooth Pairing: The car pairs with a smartphone via Bluetooth for wireless control.
  • Command Input: The user inputs directional commands through the mobile app.
  • Movement Execution: The car responds to commands by controlling its motors for movement.
  • Real-Time Control: Allows continuous control until the user decides to disconnect.
Conclusion:

The Arduino-Based Bluetooth-Supported Car project successfully demonstrates the application of wireless communication and microcontroller programming to create a functional remote-controlled vehicle. The project integrates hardware and software components to achieve real-time control over the car's movements. Future improvements could include adding sensors for obstacle detection, enhancing the mobile app interface, and incorporating additional features such as speed control and automated driving modes.

Project : Aurdino Based Wireless Notice-board
Introduction:

This project implements an Arduino-based wireless noticeboard that allows users to display messages remotely via Bluetooth and was also a part of PGC Robotics Exhibition 2022. The system uses an Arduino microcontroller and a Bluetooth module to update the noticeboard in real-time, replacing traditional manual methods.

Objective:

The objective is to design a wireless noticeboard that can be updated remotely using Bluetooth technology, demonstrating the application of microcontroller programming and wireless communication.

Key Features:
  • Wireless Communication: Uses Bluetooth for seamless wireless updates.
  • User-Friendly Interface: Messages are sent from any Bluetooth-enabled device.
  • Real-Time Updates: Messages are displayed on the noticeboard instantly.
  • Compact Design: The system is compact and easy to install.
Implementation Details:
  • Bluetooth Connection: Establishes a connection with a Bluetooth-enabled device.
  • Message Input: User sends a text message via Bluetooth.
  • Data Processing: Arduino receives and processes the message.
  • Display Update: The message is displayed on the 16x2 LCD.
  • Continuous Operation: The system remains active for further updates.
Conclusion:

The Arduino-Based Wireless Noticeboard project successfully demonstrates the integration of microcontroller programming with Bluetooth communication to create a functional and modern noticeboard system. The project can be enhanced by adding features like multiple message support and extended range communication.
