Project : Comprehensive Currency Converter
Introduction:

I submitted it as my 2nd semester Programming Fundamental's project. This project implements a console-based Currency Converter program using C++. The program allows users to convert amounts from one currency to another using predefined conversion rates. The user can choose from a list of 20 currencies and perform multiple conversions, with the results being saved to a file for record-keeping.

Objective:

The primary objective of this project is to develop a user-friendly application that facilitates currency conversion between various international currencies. Additionally, the project aims to demonstrate the use of basic C++ features such as file handling, loops, arrays, and formatted output.

Key Features:
  • User Interface: A simple console-based UI for input and output.
  • Currency List: Supports conversion between 20 different currencies.
  • Data Persistence: Records conversion results in a text file.
  • Continuous Operation: Allows multiple conversions without restarting the program.
Implementation Details:
  • Welcome Screen: Displays a welcome message.
  • User Choice: User can choose to continue with the conversion or exit the program.
  • Currency Selection: User selects the source and target currencies from a list.
  • Amount Input: User inputs the amount to be converted.
  • Conversion Calculation: Program calculates the converted amount.
  • Output: The result is displayed on the console and saved to a file.
  • Repeat or Exit: User can choose to perform another conversion or exit.
Conclusion:

The Currency Converter project successfully demonstrates the use of fundamental C++ programming constructs to build a functional application. The program handles user input, performs calculations, manages file I/O, and provides a continuous user experience. This project can be further enhanced by integrating real-time currency exchange rates through an API and adding a more sophisticated user interface.
